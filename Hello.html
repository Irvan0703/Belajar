Materi yang sudah dipelajari dalam Git

Berikut ini merupakan perintah yang ada didalam git :

  1. git init yang berfungsi untuk menginisialisi repositori yang akan digunakan pada folder local
  2. git config yang berfungsi untuk konfigurasi git. Pada git config terdapat dua konfigurasi yaitu :
    • config local yang berfungsi mengatur pengaturan email dan username local
    • config global yang berfungsi mengatur pengaturan ke server online
  3. git add berfungsi untuk memasukkan file ke dalam ruang tunggu sebelum dicommit
  4. git commit berfungsi untuk mencatat perubahan file
  5. git status berfungsi untuk mengetahui status file yg ada pada repositori git
  6. git log berfungsi untuk menampilkan riwayat perubahan yang sudah dilakukan pada repositori
  7. git remote add origin berfungsi untuk interaksi dengan repositori yang ada di server
  8. git push berfungsi untuk mengirim perubahan pada repositori local ke server
  9. git pull berfungsi untuk menerima perubahan yang dilakukan dari server ke repositori local
  10. git clone berfungsi untuk mengcopy repositori server ke repositori local
  11. git branch berfungsi untuk melihat daftar branch di repositori local
  12. git branch nama berfungsi untuk membuat branch baru
  13. git checkout nama_branch berfungsi untuk pindah ke branch lain tersebut
  14. git merge nama_branch berfungsi untuk mencopy perubahan yang sudah dibuat di branch lain
  15. git branch -d nama_branch berfungsi untuk hapus branch

Selain perintah git, kita juga mempelajari tentang conflict ketika akan commit.

Ketika ada perubahan dari file yang sama dari dua atau lebih branch maka akan mengalami conflict ketika dicommit ke repositori git. Untuk menyelesaikan permasalahan tersebut maka yang bisa dilakukan adalah membuka file yang conflict dan mengkoreksi bagian yang membuat conflict tersebut. Selain membuka file secara manual, kita juga bisa menggunakan aplikasi diffmerge yang akan menampilkan bagian mana saja yang menyebabkan conflict dan kita bisa mengubah bagian tersebut supaya bisa di commit.
